# ysh-cli

<a href="https://www.npmjs.com/package/ysh-cli"><img src="https://img.shields.io/npm/v/ysh-cli.svg" alt="Version"></a>

The CLI for scaffolding template projects, and do more things.

### install

```sh
npm i ysh-cli -g
```

### 使用方法

```sh
//初始化项目，目前可选择的模板有：react+mobx  vue+vuex
yush init
//增加模板：可以增加自己想要的模板
yush add
//删除模板
yush delete
//模板list
yush list
```
